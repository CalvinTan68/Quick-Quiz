## About Project

This is a app to test your general knowledge. It is built with :

- HTML
- CSS
- JavaScript (ReactJS)
- React-Bootstrap

## Using Project

- Execute this line

```bash
  git clone https://github.com/CalvinTan68/Quick-Quiz.git
```

- After that 

```bash
  cd Quick-Quiz
```

- Run command below to install components and dependencies 

```bash
  npm install
```

- Run command below to start

```bash
  npm start
```
